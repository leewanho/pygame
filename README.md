# 🩸 Blood Surfer (블러드 서퍼)

혈관 속을 질주하며 건강한 식재료를 수집하고 유해 요소를 피하는 **2D 아케이드 액션 게임**.
Python + Pygame 기반으로 제작되었습니다.

---

## 🎮 게임 소개

플레이어는 혈관 내부를 돌며 캐릭터를 조종합니다.
브로콜리, 블루베리, 아보카도 등 건강한 아이템을 먹어 **혈액 순도(Purity)** 와 **특수 게이지(Special Gauge)** 를 채우고,
정크푸드 · 담배 · 알코올 · 스트레스 같은 유해 요소를 회피하며 살아남는 것이 목표입니다.

## 🕹️ 조작법

| 키 | 동작 |
|---|---|
| ← / → | 좌우 이동 (혈관 회전) |
| SPACE | 특수 능력 발동 (게이지 100% 시) |
| R | 게임 재시작 |
| ESC | 일시정지 / 종료 |

## 🥦 아이템 시스템

**Good Items** (순도 + 게이지 상승)
- 브로콜리, 블루베리, 아보카도, 토마토, 아몬드

**Bad Items** (순도 감소 / 디버프)
- 정크푸드, 담배, 알코올, 스트레스

**Special Items** (특수 능력)
- 덤벨, 알약, 러닝머신

## 🛠️ 기술 스택

- **Language**: Python 3.x
- **Library**: Pygame
- **해상도**: 1000 × 1000
- **단일 파일**: `pygame_ing.py` (약 1,000 LOC)

## ⚙️ 주요 구현 포인트

- **이미지 스케일/회전 캐시** (`scaled_cache`, `rotated_cache`) — 매 프레임 변환 비용 절감
- **프리컴퓨트 아틀라스** (`build_item_atlas_for_image`) — 아이템 이미지 사전 생성으로 프레임 드랍 방지
- **원근감 기반 혈관 링 렌더링** — 중심점에서 외곽으로 확장되는 의사 3D 효과
- **동적 지방 생성** (`draw_dynamic_fat`) — 속도/위치에 따라 장애물 실시간 변화
- **다채널 사운드 관리** — BGM, SFX, 튜토리얼 음성 분리 제어

## ▶️ 실행 방법

```bash
pip install pygame
python pygame_ing.py
```
